score_genes_for_marginals(gene_set_proliferation=None, gene_set_apoptosis=None, proliferation_key='proliferation', apoptosis_key='apoptosis', **kwargs)[source]

Compute gene scores to obtain prior konwledge about proliferation and apoptosis.

This method computes gene scores using scanpy.tl.score_genes(). Therefore, a list of genes corresponding to proliferation and/or apoptosis must be passed. ALternatively, proliferation and apoptosis genes for humans and mice are saved in moscot. The gene scores will be used in moscot.problems.TemporalProblem.prepare() to estimate the initial growth rates as suggested in [Schiebinger et al., 2019]

Parameters

  • gene_set_proliferation (Union[Literal[‘human’, ‘mouse’], Sequence[str], None]) – Set of marker genes for proliferation used in the birth-death process. If marker genes from moscot are to be used the corresponding organism must be passed.

  • gene_set_apoptosis (Union[Literal[‘human’, ‘mouse’], Sequence[str], None]) – Set of marker genes for apoptosis used in the birth-death process. If marker genes from moscot are to be used the corresponding organism must be passed.

  • proliferation_key (str) – Key in anndata.AnnData.obs where to add the genes scores.

  • kwargs (Any) – Keyword arguments for scanpy.tl.score_genes().

Returns

  • proliferation_key

  • apoptosis_key

Return type

Self and updates the following attributes

Notes

The marker genes in moscot are taken from the following sources:

  • human, proliferation - [Tirosh et al., 2016].

  • human, apoptosis - Hallmark Apoptosis, MSigDB.

  • mouse, proliferation - [Tirosh et al., 2016].

  • mouse, apoptosis - Hallmark P53 Pathway, MSigDB.

prepare(time_key, joint_attr=None, policy='sequential', marginal_kwargs=mappingproxy({}), **kwargs)[source]

Prepares the TemporalProblem for it being ready to be solved

This method executes multiple steps to prepare the problem for the Optimal Transport solver to be ready to solve it

Parameters

  • time_key (str) – Key in anndata.AnnData.obs which defines the time point each cell belongs to. It is supposed to be of numerical data type.

  • joint_attr (Union[str, Mapping[str, Any], None]) – Parameter defining how to allocate the data needed to compute the transport maps. If None, the data is read from anndata.AnnData.X and for each time point the corresponding PCA space is computed. If joint_attr is a string the data is assumed to be found in anndata.AnnData.obsm. If joint_attr is a dictionary the dictionary is supposed to contain the attribute of anndata.AnnData as a key and the corresponding attribute as a value.

Return type

Self

Raises

  • KeyError – If time_key is not in anndata.AnnData.obs

  • KeyError – If joint_attr is a string and cannot be found in anndata.AnnData.obsm
